# 用户中心项目
----
## 项目介绍
----
用户中心系统：一个简单的“用户管理系统”，实现了用户注册、登录、查询等基础功能
## 技术栈
----
### 前端
主要运用阿里Ant Design生态：
- TML + CSS + JavaScript 三件套
- React开发框架
- Ant Design Pro 项目模板
- Ant Design 端组件库
- Umi开发框架
- Umi Request 请求库

### 后端
- Java编程语言
- Spring + SpringMVC + SpringBoot 框架
- MyBatis + MyBatis Plus 数据访问框架
- MySQL数据库
- jUnit 单元测试库

### 部署
- 单机配置
- Nginx
- 容器
